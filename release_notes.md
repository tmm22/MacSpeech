## Installation
1. Download MacSpeechApp.dmg
2. Open the DMG file
3. Drag MacSpeechApp to your Applications folder
4. Launch and enter your OpenAI API key

## Requirements
- macOS 11.0 or later
- OpenAI API key
- Internet connection
- Microphone access for speech-to-text
